# TimeShifter Web

一个参考 TimeShifter 使用方式的网页版倒时差原型。

## 功能

- 输入出发 / 目的地时区、出发 / 到达时间
- 根据时差方向生成逐日作息过渡计划
- 输出睡眠、光照、咖啡因、褪黑素建议窗口
- 自动把最近一次输入保存在浏览器 `localStorage`
- 纯静态前端，无需安装依赖

## 运行

直接打开 `index.html` 即可。

如果你更希望通过本地静态服务运行，也可以执行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 说明

当前版本是一个 MVP，目标是复刻这类产品的核心体验，而不是医学级个体化建议系统。
算法采用可解释的规则：

- 向东飞：逐步提前作息，强调起床后的早段光照
- 向西飞：逐步延后作息，强调下午到傍晚光照
- 咖啡因窗口会避开目标入睡时间前的较长区间
- 褪黑素建议默认只在向东飞时给出

如果后续要继续逼近原版 App，可以继续加：

- 航班号 / 中转支持
- 用户个体限制条件（失眠、晨型夜型更细分、咖啡因敏感度）
- 更精细的时区与昼夜节律模型
- PWA 安装、日历提醒、通知推送
